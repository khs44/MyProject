프로젝트 명 : 영어 학습 LMS 프로젝트
기간 / 인원 : 25.10 ~ 25.11 / 4명
개발 : Java Script, Data Base, Spring Boot
개발 툴 : SpringToolSuite4, MySQL WorkBench
주요 역할 : 프로젝트의 기획 및 DB 설계, 로그인 CRUD 제작 (고객 및 관리자 별도), 영어 학습 동영상 등록 및 삭제 기능 제작, 토스 API 테스트버전으로 결제 시스템 개발, 팀원들 코드 종합 및 오류 수정, 발표 ppt 검토 및 개발 일지 작성
